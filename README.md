#  Webpack hot boilerplate

## Contains

- [x] [Webpack 2.3](https://webpack.js.org/)
- [x] [Babel 6.x](https://babeljs.io/)

## Features

- [x] Tree shaking
- [x] Code Splitting
- [x] Hot Module Replacement
- [x] ES-2015
- [x] ejs
- [x] SCSS + postCSS
- [x] Sprites compiling

## System Requirements

- Node 6.x.x
- Npm 3.x.x

## Setup

```
$ npm install
```
or
```
$ npm install --production
```

## Running

```
$ npm start
```

## Build

```
$ npm run build
```
